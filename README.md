# My to do list

---

## 구현해야 될 기능
- Todo의 ID
- Todo의 제목
- Todo의 내용
- `이전으로` 버튼
    - `이전으로` 버튼을 구현하고, `이전으로` 버튼을 클릭하면 리스트 화면으로 되돌아 갑니다.
---

## 제한사항
- map을 사용할 때 **반드시 key**을 넣어야 하며, `map` 의 **index를 사용을 금지**합니다. 
- Todo Id 생성 시 `todos.length` 사용해서 생성하지 않습니다. 
`todos.length` 을 사용해서 id 생성 시 **발생할 수 있는 문제점에 대해 고민**해보시길 바랍니다.
